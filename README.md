# phpdesign

结合一些框架源码 以及 《深入PHP：面向对象、模式与实践》书中讲解的一些设计模式进行总结和归类

示例中的代码基于PHP7.4.3实现

---
### 设计模式基本原则

- 按接口编程而不是按实现来编程 (Program to an interface, not an implementation)
- 优先使用组合而不是继承(Favor object composition over class inheritance.)
